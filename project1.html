Design system

Toga design system

Un moyens d'accéléré le développement d'un produit.

Un design system est un procédé englobant tous les éléments concernant le point de contact entre les acteurs du développement d’un produit. Un design system a donc pour objectif de réunir tous les éléments graphiques, visuels, liés aux règles, de branding, de ton, etc. dans un seul et même système standardisé.

  • UX UI Design

    Conception d'interface et design d'expérience utilisateur web est mobile

  • Développement frontend

    Développement d'interface front-end en html, css et javascript

  • Identité graphique

    Création d'identité visuelle: logo, charte graphique, site web

Contexte & objectif

Il s'agit d'un projet sur lequel j'ai travaillé au cours de mon alternance. La création d'un design system avait été initier dans le cadre du développement d'un nouveau produit type SAS par l'entreprise (un DSP permettant l'achat d'espace en programmatique).

L'objectif était dans un premier temps d'accélérai le développement du produit en facilitant la collaboration entre product manager, développeurs et designer et dans un second temps d'assurer le maintient du produit dans le temps en répertoriant tout les compensant, code et guidelines nécessaire.

Ce design système pourra également être réutiliser par l'entreprise dans le développement de nouveau produits et permettant un onboarding plus rapide au nouveaux membres de l'équipe

Appetit: 5 mois

Entreprise: S4M France

Type de projet: Alternance

Objectif: Maintenir et accélérer
le développement d'un SAS
